
# GTA Text Alerts Bot

Бот без GTAWeb и GTALens.
Работает только на текстовых оповещениях.

## Что умеет
- хранит и показывает текстовые статусы:
  - Gun Van
  - Street Dealers
  - Stash House
  - Shipwreck
  - Hidden Caches
  - Новости
  - Бонусы недели
- админ может обновлять их командами `!set...`
- бот публикует красивое текстовое оповещение сразу после обновления

## Переменные Railway
- DISCORD_TOKEN
- CHANNEL_ID
- SEND_STARTUP_MESSAGE=true
