
# GTA Service Bot — One Message PRO

Эта версия отправляет одну большую картинку со всеми картами в одном сообщении.

## Команды
- !gta
- !gunvan
- !dealers
- !stash
- !shipwreck
- !caches
- !status

## Админ-команды
- !setgunvan x y
- !setdealers x1 y1 x2 y2 x3 y3
- !setstash x y
- !setshipwreck x y
- !setcaches x1 y1 x2 y2 x3 y3

Координаты задаются в диапазоне 0..100.
