# GTA Discord Bot (GTAWeb)

Версия бота, которая использует GTAWeb вместо GTALens.

## Что умеет
- новости GTA Online
- Gun Van
- Street Dealers
- Stash House
- Shipwreck
- Hidden Caches
- команды:
  - !gta
  - !gunvan
  - !dealers
  - !stash
  - !shipwreck
  - !caches
  - !status
  - !help

## Railway variables
Обязательные:
- DISCORD_TOKEN
- CHANNEL_ID

Необязательные:
- CHECK_INTERVAL_MINUTES=30
- SEND_STARTUP_MESSAGE=true
- ROLE_PING_ID=
- DEEPL_API_KEY=
