
# GTA PRO Map Bot

Эта версия не шлёт ссылки. Она рисует карты с метками прямо в Discord.

## Что умеет
- `!gunvan`
- `!dealers`
- `!stash`
- `!shipwreck`
- `!caches`
- `!gta` — отправляет все карты
- админ может менять координаты командами `!set...`

## Переменные Railway
- `DISCORD_TOKEN`
- `CHANNEL_ID`
- `AUTO_POST_INTERVAL_MINUTES=360`

## Как задавать координаты
Координаты в процентах от 0 до 100.

Примеры:
- `!setgunvan 62 58`
- `!setstash 44 47`
- `!setshipwreck 86 70`
- `!setdealers 26 38 51 63 74 31`
- `!setcaches 18 77 34 82 70 76`
